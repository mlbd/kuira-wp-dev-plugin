# Acme Notes

Reference plugin generated by `/wp-new` — a React admin screen backed by a secure REST API and a custom table.

## Facts
- Slug / text domain: `acme-notes`
- Prefixes: fn `acme_notes_`, class `Acme_Notes_`, const `ACME_NOTES_`
- Frontend stack: React (`@wordpress/scripts`)
- Build: `npm run build`
- Distribution: both (wordpress.org-ready + self-hosted)
- WooCommerce/HPOS: no
- Commits: manual — only when explicitly asked

## Conventions
- WordPress Coding Standards (tabs, escaping on output, sanitization on input).
- Nonce + capability checks on every REST/AJAX handler.
- Text domain `acme-notes` on every user-facing string.
- DB schema changes require bumping `ACME_NOTES_DB_VERSION` so migrations run.

## Commands
- Lint:  `composer lint`   Fix: `composer lint:fix`
- Analyze: `composer analyze`
- Test:  `composer test`  (start env: `npm run wp-env start`)
- Build: `npm run build`  Watch: `npm start`
- Preview (Docker-free): `npx @wp-playground/cli@latest server --mount=.:/wordpress/wp-content/plugins/acme-notes --blueprint=blueprint.json`
