== Description ==

Acme Notes is a small, complete reference plugin produced by the kuira-wp-dev-plugin toolkit. It demonstrates the patterns the toolkit generates:

* A React admin screen built with `@wordpress/scripts`.
* A secure REST controller (capability checks + nonce + sanitized args).
* A custom database table with `dbDelta` and version-gated migrations.
* PHPUnit tests and a WordPress Playground Blueprint.

It is intended as documentation-by-example, not as a plugin to ship.

== Installation ==

1. Upload the `acme-notes` folder to `/wp-content/plugins/`.
2. Run `composer install` and `npm install && npm run build`.
3. Activate through the 'Plugins' menu in WordPress.
4. Open the "Acme Notes" admin menu item.

== Frequently Asked Questions ==

= Is this a real plugin? =

No — it's a reference output of the kuira `/wp-new` wizard, kept in the toolkit repo so you can see exactly what gets generated.
